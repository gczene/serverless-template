## Serverless template

### Setup

First install serverless globally

```
npm install -g serverless
```

Then clone this repo and npm

```
git clone https://github.com/jholt1/serverless-template.git
cd serverless-template
npm i
```

### Localhost
localhost:3000
```
sls offline
```

or specify a port

```
sls offline --port 3001
```
